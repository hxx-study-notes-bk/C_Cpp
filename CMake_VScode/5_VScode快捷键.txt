ctrl + `：打开/关闭终端
ctrl + b：打开/关闭侧边栏
F2：变量统一重命名
alt + up/down：当前行上下移动
ctrl + tab：文件之间切换
ctrl + \：切除一个新的编辑器，最多三个
ctrl + 1/2/3：编辑器之间进行切换
alt + shift + f：代码格式化
alt + shift + 鼠标左键：列编辑
alt + shift +a：多行注释
ctrl + '+/-'：放大/缩小字体

F11：全屏
分屏：在打开的单个文件上->右键->Split Right，也可以拉着文件挪

OUTLINE：在这里可以找到当前点开的文件中定义的变量及函数
TIMELINE：修改代码的记录，可对照
SEARCH：可在整个文件夹中查找包含某个关键字的文件
